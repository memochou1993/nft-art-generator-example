nft-art-generator-example
===

## Usage

Install dependencies.

```BASH
npm install -g nft-art-generator
```

Generate images.

```BASH
nft-generate
```

Learn more about [nft-art-generator](https://github.com/NotLuksus/nft-art-generator).
